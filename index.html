Catálogo de Fauna

Sistema de fauna que muestra un listado de animales, en los cuales se visualiza información relevante, se puede filtrar por habitat. Incluye funcionalidades para administrar usuarios, roles y estados mediante operaciones CRUD. El acceso al sistema se gestiona a través de solicitudes de acceso, garantizando control y seguridad en la gestión de datos.

ASP.NET HTML JavaScript CSS Bootstrap SQL Server
  • Inicio de sesión
  • Agregar, editar y eliminar animales
  • Cambio de contraseña
  • Visualización por hábitat
  • Acceso por medio de solicitud
  • Control de acceso según rol

Gestión de proyectos

Sistema sencillo para la gestión de proyectos. Al iniciar sesión, se muestran dos gráficos interactivos: uno con la cantidad de empleados por departamento y otro con el estado actual de los proyectos y el salario promedio. El sistema permite administrar usuarios, proyectos y asignar empleados a proyectos mediante operaciones CRUD. El acceso se gestiona mediante credenciales asignadas a cada usuario, e incluye una opción de recuperación de cuenta para mayor seguridad.

ASP.NET HTML JavaScript CSS Bootstrap SQL Server
  • Inicio de sesión
  • Visualización de datos mediante gráficos
  • Agregar, editar y eliminar empleados
  • Asignación de empleados a proyectos
  • Gestión de proyectos
  • Acceso por asignación de credenciales
  • Recuperación de contraseña
